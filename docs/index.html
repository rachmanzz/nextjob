START

Open the "NextJob" bot is optional, you can use it anywhere, with friends, family or in group chats. If you Open the "NextJob" bot, you will find another option like filter menu and guide.

USAGE

Open any chat, and type @nextjobot [space] position (job you looking for)

example:

@nextjobot backend

INLINE FILTER

To add filter we need extra argument to filter result.

optional filter

  1. loc

    loc represents location. multi location is able. to add multi location, split location value by semicolon (;)

  2. salary

    salary format [start_salary] - [stop_salary] for example 200-800

  3. currency

    currency code

  4. type

    type of job as full-time, part-time, contract, etc

example:

@nextjobot backend --loc=indonesia --currency=usd

MULTI POSITION

this bot able search multi position by split with semicolon (;)

example:

@nextjobot backend; frontend

SETUP FILTER

open the "NEXTJOB" bot chat, and type /filter or open menu and select filter. if you have filter set, you not need to use inline filter. filter will saved 8 hours.
